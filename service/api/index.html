Relabel Web Service API

This WS was created to provide detailes about
our system's users and their profile settings, also providing
mathcing tracks by the user's preferences.
As a RESTful Serveice all users & track's detailes will be returned
in JSON format.

Login: [POST]

https://relabel--service.herokuapp.com/users/login/

Parameters:

[POST]
username : string
password : string

JSON Response Example:

{ "_id" : "5948ee45f36d28648d054faa", "username" : "Nivo", "token" : "eyJhbGciOiJIUzI" }

Get Preferences by Id: [POST]

https://relabel--service.herokuapp.com/users/getPrefById/

Parameters:

[POST]
id : string - currently pass username :)

JSON Response Example:

[ { "name": "detriot", "value": 3 }, { "name": "hard", "value": 0 }, { "name": "dance", "value": 0 }, { "name": "minimal", "value": 7 }, { "name": "classic", "value": 1 }, { "name": "house", "value": 0 }, { "name": "vgm", "value": 0 }, { "name": "hard_acid", "value": 0 }, { "name": "electro", "value": 0 } ]

Get Tracks by Preferences: [POST]

https://relabel--service.herokuapp.com/tracks/getTracksByPref/

Parameters:

[POST]
id : string - currently pass username :)

JSON Response Example:

[ { "_id": "59462c52734d1d59b7894817", "name": "Strings of Life", "artist": "Rhythim Is Rhythim", "album": "Extrawelt v", "year": 1987, "length": 707, "genre": [ "detriot" ] } ]
